GDT: Game Data Template
for movement actions
parameters:
0=ActiveCharacter
1=Action.Name.Past
2=Action.Name.Present
3=Action.Direction
4=LevelNumber
5=ObjectsBlocking
6=CharactersBlocking

for other actions:
0=ActiveCharacter
1=Action.Name.Past
2=Action.Name.Present
3=TargetObject
4=TargetCharacter
5=ActiveObject
